# Palindrome Number

---

# What is a Palindrome Number?

A Palindrome Number is a number that remains the same when its digits are reversed.

In simple terms, it reads the same from left to right and from right to left.

Examples

121 → Reversed: 121 → Palindrome

1331 → Reversed: 1331 → Palindrome

123 → Reversed: 321 → Not a Palindrome

---

# Logic Steps:
- Input: An integer number
- Store the original number
- Reverse the digits using modulus and division
- Compare the reversed number with the original number
- If both are equal → Palindrome
- Otherwise → Not a Palindrome
