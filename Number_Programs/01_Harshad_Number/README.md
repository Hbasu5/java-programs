# What is a Harshad Number?

A Harshad number (or Niven number) is an integer that is divisible by the sum of its own digits in a given base, usually base 10. The core logic involves calculating the sum of the digits and performing a modulus operation to check for a zero remainder.

---

# Logic Steps:
- Input: An integer (Example - 156)
- Sum of Each Digit: 1+5+6 = 12
- If Input number divisible by sum of digits, then number is a Harshad Number else not
