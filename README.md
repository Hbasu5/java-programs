# Java Programs
---

# Overview:

This repository contains a structured collection of fundamental Java programs developed as part of academic training and logical problem-solving practice.

The programs focus on strengthening core programming concepts and mathematical reasoning using basic Java constructs.

---

# Repository Structure:

- Number Programs – Digit-based and number theory programs
- Array Programs – Sorting, searching, and array manipulation
- String Programs – String processing and transformation

---

# Concepts Covered:

- Conditional statements
- Loops and nested loops
- Methods and structured logic
- Arrays
- Strings
- Basic object-oriented structure

---

# Objective

The purpose of this repository is to document foundational Java practice and demonstrate progressive improvement in structured programming and problem-solving skills.

---

# How to Run:

- Compile any program using: javac <"PROGRAM_NAME">.java
- Run using: java <"PROGRAM_NAME">
